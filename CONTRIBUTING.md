# Contributing

CloudTime is a personal project. Contributions are limited to the following:

## Welcome

- **Bug reports** — Open an issue with steps to reproduce
- **Feature requests** — Open an issue describing the use case

## Not accepted

- Pull Requests without a prior approved issue
- Code style or refactoring-only changes
- Documentation translation or cosmetic PRs

If you'd like to propose a bug fix or feature via PR, **open an issue first and wait for approval**. Unsolicited PRs will be closed.
